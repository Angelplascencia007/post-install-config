<h1>osTicket - Post-Install Configuration</h1>
This tutorial outlines the post-install configuration of the open-source help desk ticketing system osTicket.<br />

- ### [YouTube: How To Configure osTicket, post-installation](https://www.youtube.com)

<h2>Environments and Technologies Used</h2>

- Microsoft Azure (Virtual Machines/Compute)
- Remote Desktop
- Internet Information Services (IIS)

<h2>Operating Systems Used </h2>

- Windows 10</b> (21H2)

<h2>Post-Install Configuration Objectives</h2>

-  Configure Roles

- Configure Departments
- Configure Teams
- Allow Anyone to Create Tickets
- Configure Agents
- Configure Users
- Configure SLAs
- Configure Help Topics

<h2>Configuration Steps</h2>

This tutorial outlines the post-install configuration of the open-source help desk ticketing system osTicket.

<P>1. Configure Roles
Navigate to the Admin panel.
Select "Agents" then "Roles".
Add new roles and assign permissions as needed.
<P><img width="334" alt="image" src="https://github.com/Angelplascencia007/post-install-config/assets/168943120/ef38aeb5-8f6d-44d3-84b3-935d9dc294f5">



<P>2. Configure Departments
Navigate to the Admin panel.
Select "Agents" then "Departments".
Create new departments and configure settings.
<P><img width="334" alt="image" src="https://github.com/Angelplascencia007/post-install-config/assets/168943120/986b355a-74d3-4f27-8f21-356317af5b50">


<P>3. Configure Teams
Navigate to the Admin panel.
Select "Agents" then "Teams".
Add new teams and assign agents to teams.
<P><img width="334" alt="image" src="https://github.com/Angelplascencia007/post-install-config/assets/168943120/fc95696f-be63-4d5d-8ed4-08e9e54acc3b">


<P>4. Allow Anyone to Create Tickets
Navigate to the Admin panel.
Select "Settings" then "Tickets".
Ensure "Client Quick Access" is enabled to allow public ticket creation.
<P><img width="333" alt="image" src="https://github.com/Angelplascencia007/post-install-config/assets/168943120/57d1827d-37d8-4f27-8f20-e3099d8bf835">

<P>5. Configure Agents
Navigate to the Admin panel.
Select "Agents" then "Add New Agent".
Fill in the necessary details to create agent accounts.
<P><img width="332" alt="image" src="https://github.com/Angelplascencia007/post-install-config/assets/168943120/f5fd28e8-0312-4ba5-8a90-bf6cb2f1472e">

<P>6. Configure Users
Navigate to the Admin panel.
Select "Users" then "Add New User".
Fill in the necessary details to create user accounts.
<P><img width="224" alt="image" src="https://github.com/Angelplascencia007/post-install-config/assets/168943120/88f5b13c-e1ee-4e1a-b980-50df4037be30">

<P>7. Configure SLAs
Navigate to the Admin panel.
Select "Manage" then "SLA Plans".
Create and define SLA plans as needed.
<P><img width="332" alt="image" src="https://github.com/Angelplascencia007/post-install-config/assets/168943120/db6a96ec-2301-4744-bd05-fa545aeb47ab">

<P>8. Configure Help Topics
Navigate to the Admin panel.
Select "Manage" then "Help Topics".
Add new help topics to categorize ticket issues effectively.
<P><img width="332" alt="image" src="https://github.com/Angelplascencia007/post-install-config/assets/168943120/2460364f-167b-4f36-8e87-fd6095d8f08d">

<P>By following these steps, you will have a well-configured osTicket system ready to manage help desk operations efficiently.
